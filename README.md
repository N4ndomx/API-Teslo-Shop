## Installation

```bash
$ npm install
```
# Configurar .ENV

<p>Renombrar ".env.template" a ".env" y colocar las variabes correspondientes segun el entorno </p>

## Docker-Compose

```bash
docker-compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
